# physics-g-cards handoff kit

このZIPは、物理受験カード v2 の引き継ぎ用ファイルです。

## 含まれるファイル

- `HANDOFF.md`  
  プロジェクトの正本。新しいChatGPTスレッドでは最初にこれを読ませる。

- `CHANGELOG.md`  
  変更履歴。何を決めたか、何を直したかを残す。

- `TODO.md`  
  次にやること。

- `NEXT_THREAD_PROMPT.md`  
  新しいChatGPTスレッドを始める時に貼る文章。

## 使い方

1. このZIPを解凍する
2. `HANDOFF.md`, `CHANGELOG.md`, `TODO.md` を GitHub の `physics-g-cards/` フォルダへ置く
3. 新しいスレッドでは `NEXT_THREAD_PROMPT.md` の内容を貼る
4. 必要に応じて `HANDOFF.md` を添付する

巨大な `index.html` をチャットに貼る必要はありません。
GitHub上の公開ファイルを正本にして進めます。
