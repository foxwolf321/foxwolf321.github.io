このフォルダは「私立理系 受験物理 基本法則カード（力学編）」のPWA一式です。

入っているファイル:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- README.txt

置き方:
- GitHub Pages などで、このフォルダごと配置してください。
- どのフォルダ名でも動くように、manifest と service worker は相対パスで作ってあります。

使い方:
1. index.html を開く
2. カード学習・一覧・説明書を切り替える
3. ホーム画面に追加して使う
4. 進捗はブラウザ内に保存されます
5. 必要なら「進捗を書き出す」でJSON保存できます
